# The "Chaos Monkey" E-Commerce Sandbox

The Product: A microservices-based e-commerce store (cart, checkout, inventory). The actual project is an
automated chaos engineering pipeline that randomly kills pods or degrades network latency to test system
resilience.

Core Architecture: Kubernetes, Istio (Service Mesh), Chaos Mesh or Gremlin, multiple dummy microservices.

The Portfolio Flex: Proves deep Kubernetes expertise. Shows you know how to configure Horizontal Pod
Autoscalers (HPA), circuit breakers, and Istio retries to ensure a user's checkout doesn't fail even if the inventory database pod is suddenly deleted.
